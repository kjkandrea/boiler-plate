## Git

### 이미 Tracked 되고있는 디렉토리 & 파일 제거하는법

.gitignore 추가했는데 tracked에서 안 빠질때 화가난다 화가나 하지말구 아래 명령어를 입력해보자.

``` bash
git rm --cached target_directory -r 
```
